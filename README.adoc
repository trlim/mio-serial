= Serial port I/O for mio

This crate provides extension to David Cuddeback's `serial` for simpler integration
with `mio` event loop.

== Usage

`Cargo.toml`
----
[dependencies]
serial = "0.3"
serial-mio = "0.1"
----

== Compatibility

The following platforms are confirmed to be compatible:

* macOS (x86_64)
